== Description ==

Une extension WooCommerce permettant d'ajouter les villes et regions du Sénégal.

Permet de mieux fixer les  forfaits des taux d’expédition
mieux gerer les classes d’expédition 

Si votre boutique WooCommerce doit calculer les frais d’expédition en fonction de la zone 
de livraison pour le Sénégal, alors cette extension est faite pour vous.

== Frequently Asked Questions ==

= Peut on utiliser cette extension pour les autres pays africain =

Pas pour les moments, nous allons ajouter les autres pays au fure et à mésure.

== Feature ==

1. Forfait d’expédition basés sur  la zone de livraison
2. Livraison GRATUITE dans les tranches de zone de livraison sélectionnées
3. WooCommerce 3.0 et supérieure
4. Frais de traitement comme taux fixe ou comme pourcentage 


== Installation ==
Just go to WordPress marketplace plugin and search for SN Zone to use this plugin for WooCommerce add-on.
